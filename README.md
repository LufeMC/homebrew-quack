# Homebrew Tap for Quack

Zero-config bug detection for your CI.

## Install

```bash
brew tap LufeMC/quack
brew install quack
```

Or in a single command:

```bash
brew install LufeMC/quack/quack
```

## Usage

```bash
quack login       # Authenticate with your Quack account
quack init        # Initialize Quack in your project
quack scan        # Run a bug detection scan
quack scan --ci   # Run in CI mode
```
